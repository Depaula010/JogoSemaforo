# Sons do Jogo Genius

Esta pasta contém os arquivos de som para cada botão do jogo.

## Arquivos de Som

- `red.mp3` - Som do botão vermelho (nota E - 329.63 Hz)
- `green.mp3` - Som do botão verde (nota C# - 277.18 Hz)
- `blue.mp3` - Som do botão azul (nota A - 220.00 Hz)
- `yellow.mp3` - Som do botão amarelo (nota E baixo - 164.81 Hz)

## Como Gerar os Sons

Você pode gerar os sons de duas formas:

### Opção 1: Usar um Gerador Online
1. Acesse: https://www.szynalski.com/tone-generator/
2. Configure cada frequência e baixe como MP3

### Opção 2: Usar Audacity (Grátis)
1. Baixe Audacity: https://www.audacityteam.org/
2. Gerar > Tom...
3. Configure:
   - Forma de onda: Senoidal
   - Frequência: (veja acima)
   - Amplitude: 0.8
   - Duração: 0.3 segundos
4. Exportar como MP3

### Opção 3: Sons Temporários
Por enquanto, o jogo usa sons sintéticos gerados programaticamente.
Você pode substituir por arquivos MP3 reais depois.

## Notas Musicais Usadas

Baseado no Genius original:
- Vermelho: Mi (E4) - 329.63 Hz
- Verde: Dó# (C#4) - 277.18 Hz  
- Azul: Lá (A3) - 220.00 Hz
- Amarelo: Mi (E3) - 164.81 Hz
